## 🎓 Planos de Aprendizado

### 📱 Plano: "Quero Editar a Página"

**Tempo total:** 30 minutos

1. Leia: [GUIA_RAPIDO.md - Instalação](GUIA_RAPIDO.md#1️⃣-instalação-5-minutos)
2. Execute: `npm install && npm run dev`
3. Leia: [GUIA_RAPIDO.md - Editar Conteúdo](GUIA_RAPIDO.md#2️⃣-editar-conteúdo)
4. Experimente fazer as mudanças
5. Pronto! 🎉

### 👨‍💻 Plano: "Quero Contribuir com Código"

**Tempo total:** 2-3 horas

1. Faça plano anterior (30 min)
2. Leia: [GUIA_TECNICO.md - Arquitetura](GUIA_TECNICO.md#🏗️-arquitetura) (30 min)
3. Leia: [GUIA_TECNICO.md - Padrões](GUIA_TECNICO.md#📝-padrões-de-código) (45 min)
4. Experimente criar um novo componente (30 min)
5. Leia: [GUIA_TECNICO.md - Git Workflow](GUIA_TECNICO.md#📦-git-workflow) (15 min)
6. Pronto para fazer um Pull Request! 🚀

### 🏢 Plano: "Quero Publicar o Site"

**Tempo total:** 1-2 horas

1. Faça plano "Contribuir com Código" (2-3 horas)
2. Customize o site conforme seu ministério (1-2 horas)
3. Leia: [DOCUMENTACAO.md - Deploy](DOCUMENTACAO.md#🌐-deploy) (20 min)
4. Escolha uma opção (Vercel recomendado)
5. Configure domínio e SSL
6. Site ao vivo! 🌟

### 🛠️ Plano: "Setup Completo da Equipe"

**Tempo total:** 3-4 horas

1. Prepare um computador com os pré-requisitos
2. Execute: [CONFIGURACAO.md - Checklist](CONFIGURACAO.md#✅-checklist-de-setup)
3. Configure VS Code: [CONFIGURACAO.md - VS Code](CONFIGURACAO.md#-dicas-de-produtividade)
4. Crie documentação customizada para sua equipe
5. Faça workshop com a equipe (explicar os 4 docs principais)
6. Equipe pronta! 👥
