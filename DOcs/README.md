# Semeando a Palavra — Projeto base

Passos rápidos:

1. Instale Node (use nvm-windows recomendado) — instruções no documento principal.
2. Crie o projeto com `npm create vite@latest semeando-a-palavra -- --template react-ts` ou use o scaffold já presente.
3. `cd semeando-a-palavra` e `npm install`.
4. Coloque seu logo em `src/assets/logo.png`.
5. Rode `npm run dev` e abra `http://localhost:5173`.
